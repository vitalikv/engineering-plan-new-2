# engineering-plan-new-2

3-страничный сайт на Vite + TypeScript (клон engineering-plan-new без PHP).

## Страницы

- **Главная** (index.html) — 3D сцена, проекты, калькулятор блоков
- **Инструкция** (/documentation) — видео-уроки Rutube
- **Обратная связь** (/contact) — форма с мок-отправкой

## Запуск

```bash
npm install
npm run dev
```

Сайт: http://localhost:5173/

## Сборка

```bash
npm run build
```

Результат в `dist/`.

## Технологии

- Vite 7
- TypeScript
- Three.js (3D сцена)
- Vanilla TS (без фреймворков)
